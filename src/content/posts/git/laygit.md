**short cut for lazygit**
```
1. <Space> g g   (Open lazygit)
2. <Space>       (Send some files to staging area)
3. c             (Do commit)
4. P             (Do push)
5. <Space> e     (Open directory tree)
6. <C-h>         (move to directory tree)
7. <C-arrow key> (Expand window)
8. :Lazy <CR> U  (Update Lazynvim)
```
