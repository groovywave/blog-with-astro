```
Edit a macro
---
/tip 7 1 1 <CR>
qq yyp q u 　(Record yyp into register q)
:put q <CR>　(Paste registered command yyp )
A p <Esc>　　(Edit the command)
V "qy uu     (Rerecord the edited command into register q  )
@q           (Run command register q)
---
tip711
```
